Manuel de PlayListGenerator

Version 0.9.1

Description:

PlaylistGenerator est une application libre de création et de gestion de liste de lecture, écrite par Renaud GUEZENNEC en C++ utilisant le framework Qt4 et des patrons de conceptions (Design Patterns). Cette application est le projet principal de l'année d'étude de l'auteur pour le Dundalk Institute of Technology, 2007-2008, Irlande.

Fonctions:

Puissance:


  • Cettte application peut manager et charger plusieurs formats de listes de lectures (comme .pls et  .m3u).
  • Elle peut écrire une liste de lecture sous différent formats: XML, HTML, PDF, CSV, M3U et PLS.
  • La création de la liste de lecture peut être faite par une exploration automatique d'une hiérarchie de répertoire.
  • Certaines étiquettes peuvent être modifier.   
  • L'utilisateur peut changer dynamiquement les informations qu'il souhaite lire.
  • L'application fournit une fonction de tri et de recherche.

Cross-platform:


PlayListGenerator est écrite en C++ en utilisant Qt4, cela signifie qu'elle est cross-platform. Elle peut être utilisé sous GNU/Linux, Microsoft Windows et Apple Mac Os.
PlayListGenerator doit être recompiler sur chaque système pour être fonctionnel.

Internationalization :


PlayListGenerator est traduit en deux langues: anglais et français mais si vous désirer traduire cette application dans une autre langue. Envoyez-moi un courriel: renaud.guezennec@gmail.com
