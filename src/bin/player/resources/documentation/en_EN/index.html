PlayListGenerator Manuel 

Version 1.0

Overview:

PlaylistGenerator is an open source application to create and to manage a MP3 playlist, written by Renaud GUEZENNEC in C++ using Qt4 and Design Patterns. This application is the main year project of the author for Dundalk Institute of Technology, 2007-2008, Ireland.

Features:

Powerful:
  • This application can manage and load many playlist format (such as pls and m3u).
  • It can write the playlist in different formats as XML, HTML, PDF, CSV, M3U, PLS.
  • The playlist creation can be done by an automatic exploration of an directory tree.
  • Many song tags can be amended by PlayListGenerator.   
  • The user can change dynamiclly which informations, he wants to read.
  • The application provides a sort and a research.
Cross-platform:

PlayListGenerator is written in C++ using Qt4, that means it is cross-platform. We can use it on GNU/Linux, Microsoft Windows and Apple Mac Os.
PlayListGenerator must be compilated on each system to be use.

Internationalization:

PlayListGenerator is in two languages: English and French but if you want to translated it in an other language. Send me an email renaud.guezennec@gmail.com
